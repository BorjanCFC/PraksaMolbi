# PraksaMolbi - Систем за електронско поднесување молби

Веб апликација за студентско учеште - система за поднесување и управување со молби до студентска служба.

## Карактеристики

✅ **Студентски функции:**
- Регистрирање и најава
- Креирање нови молби со наслов и опис
- Преглед на своите молби по статус
- Детални преглед на секоја молба и фидбек од администратор

✅ **Администраторски функции:**
- Администраторски панел за преглед на сите молби
- Филтрирање молби по статус и студент
- Управување со статус на молби (Во процес, Одобрена, Одбиена)
- Додавање повратна информација/причина за одбивање

✅ **Технолошки опции:**
- Node.js + Express.js за бэкенд
- SQLite за база на податоци
- EJS за темплати
- Sequelize.js за ORM
- Session management за аутентификација
- bcryptjs за хеширање на лозинки

## Инсталација и стартување

### 1. Инсталирајте зависности
```bash
npm install
```

### 2. Иницијализирајте базата со пример-податоци
```bash
npm run seed
```

### 3. Стартувајте аплликација
```bash
npm start
```

## Функции

### За студенти:
1. Регистрирање и најава
2. Преглед своих молби со филтрирање
3. Креирање нова молба (наслов + опис)
4. Реглед детали молба и администраторскиот фидбек

### За администратор:
1. Администраторска најава
2. Преглед сите молби со филтрирање
3. Обработка молбе (промена статус)
4. Додавање причина за одбивање или одобрување

## Статуси
- 🟡 **Во процес** - молба чека администратор
- 🟢 **Одобрена** - молба е одобрена
- 🔴 **Одбиена** - молба одбиена со причина
